## PYBLISH_GUI

Set which GUI to register and load.

<br>
<br>
<br>

### Usage

Setting this environment variable will enable Pyblish to register and load a GUI.

<br>
<br>
<br>

### Example

```python
import os
os.environ["PYBLISH_GUI"] = "pyblish_qml"
```
